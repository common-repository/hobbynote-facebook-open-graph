Aider les utilisateurs à implémenter et personnaliser les balises Open Graph facilement.

== Description ==

Ce plugin permet d'implémenter et de personnaliser facilement les balises Open Graph sur votre blog.

Le module va ajouter directement les balises meta nécessaires à l'implémentaton du protocole Open Graph. 
Vous pourrez définir dans le backoffice du module l'id de l'application Facebook ou la liste des id Admins, les champs titre et description (pouvant être récupérés directement à partir des données entrées par le plugin WPSEO by Yoast ou ALL In One SEO), une image pour votre Homepage et vos posts.
 
Vous trouverez également 2 liens dans le menu du plugin :
- Documentation : Lien redirigeant vers la documentation de l'Open Graph.
- Debuger : Lien redirigeant vers un outil développé par notre agence. Il s'agit d'un outil nommé "SMO Analyzer" qui permet d'analyser la présence des balises Open Graph sur votre site. 

== Installation ==

Pour installer le plugin, il faudra vous rendre dans le backoffice de votre blog, vous rendre dans la catégorie "Extensions", cliquer sur Ajouter et uploader le module.
N'oubliez pas de définir l'id de l'application Facebook ou la liste des id Admins.

== Frequently Asked Questions ==

= Comment configurer le plugin ? =

Il faut se rendre dans le menu Facebook OG dans votre backoffice sur la gauche et compléter les différentes sections.
